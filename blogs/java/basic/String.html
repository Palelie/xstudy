    # String详解

    # 一、基本信息

    1. java.lang.String是java JDK内置的一个类
    2. String表示字符串类型，属于引用数据类型，不属于基本数据类型
    3. 在java中随便用双引号括起来的都是String对象
    4. java中规定，双引号括起来的字符串都是不可改变的
    5. 在JDK中双引号括起来的字符串都是直接存储在 "方法区" 的 "字符串常量池中" （字符串使用的太频繁，这样可以提高执行效率）
    6. 垃圾回收器不会释放常量（包括字符串常量池）
    //二者本质上不同
int i = 100;		//i变量保存的是100
String s = "abc";		//s引用保存的是地址，而不是"abc"

    # 二、String类型比较中"=="和equals比较的区别

    # 可以用"=="进行比较的情况

    String s1 = "hello";	//这个"hello"存储在字符串常量池中
String s2 = "hello";	//这个"hello"不会新建，s2和s1都指向同一块内存

System.out.println(s1 == s2); //true

    # 需要用equals()比较的情况

System.out.println(s1 == s2); //false	（只能比较s1和s2指向的地址）
System.out.println(s1.equals(s2)); //true	（String类的equals方法已经默认重写）

    # 三、equals的推荐书写形式（String类中equals已经重写）
