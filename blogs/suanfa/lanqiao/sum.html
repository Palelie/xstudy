    # 部分算法归纳

    # 一、判断闰年

    闰年：对于非100倍数的年份，是4的倍数就是闰年。对于是100倍数的年份，需要是400的倍数才是闰年

    另一种记忆方式：

    ​ （4位数年份）若后两位是00则看前两位是不是4的倍数，例如1900不是闰年。否则看后两位数是不是4的倍数就行。

    代码：

    bool run(int y){
    return (y%100!=0&&y%4==0)||(y%400==0);
}

    # 2、reverse()

    逆序（反转）无论是在C或是C++中用的都特别多，常用于数组，字符串，容器等，其本身的函数参数也不复杂。

    reverse函数用于反转在[first,last)范围内的顺序（包括first指向的元素，不包括last指向的元素），reverse函数没有返回值

    #include<algorithm>

vector<int> v = {5,4,3,2,1};
reverse(v.begin(),v.end());//v的值为1,2,3,4,5

string str="www.mathor.top";
reverse(str.begin(),str.end());//str结果为pot.rohtam.wwww
